
MineFantasia
=======
![image](https://github.com/Seikai-Takenawa/MineFantasia/blob/master/src/main/resources/icon.png)

Introduction:
============
-This is a Minecraft mod that allows you to play various musical instruments alone or with your friends.
-The mod animations are based on GeckoLib5.
-There are two versions of the mod:
--A common version that don't integrate GeckLib.
--Another version integrated the latest GeckLib, which means that there is no need for you to add stand alone GeckLib mod.

Instruments:
-Piano:Unbreakable block, netrurally generated in Concert Hall Structure.
-Violin:
-Harp
-Kalimba
-Flute
-Shaker
-Drum
-various kinds of synthesizers

Notes:All instruments can only be get from Concert Hall Structure.

Notice & Warning:
=========
-This project is not open source.
-If you add the standalone GeckoLib mod after using the integrated version of the mod, it may cause serious conflicts and compatibility issues！
